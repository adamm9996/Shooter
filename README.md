# Shooter
A first person shooter in C++, using OpenGL, GLEW, SDL2, glm, & SOIL.

#Controls
WASD movement

SPACE  jump

LSHIFT down

LMB    fire gun

RMB    launch grenade

MOUSE  move camera

Note - This project was originally adapted from an OpenGL tutorial, so it is written horribly.
For a more well-designed version, see Shooter2.
